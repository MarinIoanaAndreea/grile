Cisco Modul 10 - Configurarea de Baza a Routerului

Grile interactive cu raspuns corect si explicatii.

Scor: 0/13
Intrebarea 1
Un router porneste si intra in modul setup. Care este motivul?
Explicatie: Daca un router nu poate localiza fisierul startup-config in NVRAM, va intra in modul setup pentru a permite introducerea configuratiei de la consola.
Intrebarea 2
Ce comanda este folosita pentru a cripta toate parolele din fisierul de configurare al unui router?
Explicatie: Comanda service password-encryption cripteaza toate parolele din fisierul de configurare curent. Comanda enable secret cripteaza doar parola pentru modul Privileged EXEC.
Intrebarea 3
Politica companiei necesita folosirea celei mai sigure metode pentru protejarea accesului la modul privileged EXEC si de configurare. Parola este trustknow1. Care comanda ofera cel mai inalt nivel de securitate?
Explicatie: Comanda enable secret foloseste un algoritm de criptare MD5 puternic, spre deosebire de enable password care foloseste o criptare slaba tip 7, sau service password-encryption care cripteaza toate parolele din configuratie dar cu algoritm slab.
Intrebarea 4
Care va fi raspunsul routerului dupa introducerea comenzii "router(config)# hostname portsmouth"?
Explicatie: Comanda hostname schimba promptul routerului la noul hostname. Deoarece comanda a fost introdusa in Global Configuration mode (config), promptul va fi portsmouth(config)#.
Intrebarea 5
Un administrator configureaza un nou router pentru a permite accesul de management out-of-band. Ce set de comenzi va permite autentificarea cu parola cisco?
Explicatie: Accesul out-of-band (in afara benzii) se refera la accesul prin portul de consola fizica. Aceasta necesita configurarea liniei con 0 cu o parola si comanda login.
Intrebarea 6
Ce comanda poate fi folosita pe un router Cisco pentru a afisa toate interfetele, adresa IPv4 atribuita si statusul curent?
Explicatie: Comanda show ip interface brief afiseaza un rezumat al tuturor interfetelor, inclusiv adresa IP si statusul (up/down). Comanda show ip route afiseaza tabelul de rutare. Show interface fa0/1 afiseaza detalii doar pentru o interfata specifica.
Intrebarea 7
Ce mod CLI permite utilizatorilor sa acceseze toate comenzile dispozitivului, inclusiv cele de configurare, management si depanare?
Explicatie: Modul Privileged EXEC (promptul #) ofera acces la toate comenzile routerului. Modul User EXEC (promptul >) permite doar comenzi limitate de monitorizare.
Intrebarea 8
Care este scopul fisierului de configurare de pornire (startup configuration) pe un router Cisco?
Explicatie: Fisierul startup-config este stocat in NVRAM si contine configuratia folosita la pornirea routerului. Running-config este configuratia activa stocata in RAM.
Intrebarea 9
Care caracteristica descrie gateway-ul implicit al unui calculator host?
Explicatie: Gateway-ul implicit este adresa IP (logica) a interfetei routerului din aceeasi retea cu hostul. Hostul trimite pachetele destinate altor retele catre aceasta adresa.
Intrebarea 10
Care este scopul comenzii banner motd?
Explicatie: Comanda banner motd (Message of the Day) afiseaza un mesaj tuturor utilizatorilor care se conecteaza la dispozitiv. Este folosita adesea pentru avertismente legale.
Intrebarea 11
Un tehnician configureaza un router pentru toate formele de acces de management. Ca parte a fiecarui tip de acces, tehnicianul incearca sa introduca comanda login. Ce mod de configurare trebuie sa fie accesat pentru aceasta sarcina?
Explicatie: Comanda login este configurata in modul de configurare a liniei (line con 0, line vty 0 4, etc.) pentru a activa autentificarea cu parola pe acea linie.
Intrebarea 12
Ce este stocat in NVRAM-ul unui router Cisco?
Explicatie: NVRAM (Non-Volatile RAM) stocheaza fisierul startup-config. Cisco IOS este stocat in Flash. Running-config este in RAM. Instructiunile de boot sunt in ROM.
Intrebarea 13
Care afirmatie privind comanda service password-encryption este adevarata?
Explicatie: Comanda service password-encryption se introduce in Global Configuration mode si cripteaza imediat toate parolele existente in configuratie. Odata criptate, parolele nu pot fi recuperate in plaintext prin no service password-encryption.
